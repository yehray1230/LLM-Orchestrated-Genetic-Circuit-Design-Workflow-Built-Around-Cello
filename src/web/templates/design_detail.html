{# ── BOM CSV ── #}

📡

📋
JSON 完整 DesignIR v2 結構化資料，含所有欄位、假設與來源追蹤。適合程式化存取。 格式：application/json · v2 下載 JSON

💻

{# ── Plasmid GenBank ── #}
🧫

下載 Plasmid GenBank
